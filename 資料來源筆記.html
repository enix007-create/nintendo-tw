任天堂台灣 Switch / Switch 2 — 階段一偵察結果（2026-05-29）

結論摘要

✅ 可行。可在不破壞反爬蟲規範下，靠官方資料源組出完整即時清單：
  • 用 Playwright 渲染兩個目錄頁拿 NSUID + 封面 URL
  • 用 Nintendo eShop Price API 批次查 TWD 原價、特價、特價區間（公開 endpoint）
  • 用 遊戲詳情頁標題 拿中文名稱（含英文副標）

2. NSUID 萃取方式

NSUID（任天堂全球唯一商品 ID，14 位數字）藏在目錄頁的封面圖 URL 中：

https://images.ctfassets.net/o5v89n4kg6h4/70010000095685e0ee4e46894311f0909071ba1e81c7be/.../cover.jpg

正則：

images\.ctfassets\.net/[^/"\s]+/(\d{14})[a-f0-9]*/[^"\s)]+

NSUID 開頭規則（觀察值，非官方）：

  • 70010000xxxxxx — Switch 遊戲為主
  • 70070000xxxxxx — 部分 Switch 2 遊戲
  • 但兩種前綴在 Switch / Switch 2 目錄頁都會出現，不可只靠前綴判斷平台，要以「該 NSUID 出現在哪個目錄頁」為準

3. Price API 規格

Endpoint 
GET https://api.ec.nintendo.com/v1/price?country=TW&lang=zh&ids=NSUID1,NSUID2,...
實測回應（NSUID 70010000124849，有特價） 
{
  "personalized": false,
  "country": "TW",
  "prices": [{
    "title_id": 70010000124849,
    "sales_status": "preorder",
    "is_download_ready": true,
    "regular_price": { "amount": "TWD 110", "currency": "TWD", "raw_value": "110" },
    "discount_price": {
      "amount": "TWD 88",
      "currency": "TWD",
      "raw_value": "88",
      "start_datetime": "2026-05-27T16:00:00Z",
      "end_datetime":   "2026-06-28T15:59:59Z"
    }
  }]
}

sales_status 觀察到的值：onsale（販售中）、preorder（預購）、unreleased（未發售）、not_found（NSUID 在 TW 區無資料）。

特價百分比 = (1 - discount_price.raw_value / regular_price.raw_value) × 100，僅 discount_price 存在時計算。

5. 已知問題

⚠️ 待解決
  1. 遊戲總數可能不足：目錄頁滾動到底也只拿到 79 個 NSUID，但 Switch 平台應該有上千款遊戲。需要找：
    • (a) 是否有分頁／全列表 API 端點（可能藏在 Contentful CDN）
    • (b) 「全部遊戲」的隱藏入口（可能在「遊戲尋找器」之類的頁面）
  2. 英文名稱來源未定：任天堂台灣站只有中文標題。可選方案：
    • (a) 從中文標題抽英文副標（部分遊戲含英文，例如「TOMODACHI LIFE 朋友收集 夢想生活」）
    • (b) 用同一個 NSUID 去 Nintendo HK 英文版抓 (store.nintendo.com.hk/en/)
    • (c) 接受只有中文，搜尋仍可中英混合 fuzzy（Fuse.js 對任何字串都能比對）
  3. 詳情頁價格 JS 渲染：盒裝/下載 TWD 在初始 HTML 中沒有。實作時用 Playwright 即可解決，或直接信任 Price API（建議後者）
  4. 反爬蟲：目前未遇到。建議仍加合理延遲（每請求 0.3~1 秒）與正常 User-Agent
